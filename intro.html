Aplicativos

WebCode



WebCode é uma IDE voltada para linguagens web (Html, CSS e JavaScript) que oferece ao programador uma emulação de um navegador, fazendo com que ele não precise de um servidor rodando seu codigo para acompanhar processo.
Infelizmente nem tudo que é bom é de graça, sua licença está custando atualmente R$20,99 na loja do Google. Logo a baixo você encontrará o link para o aplicativo e poderá baixa-lo e usar o seu periodo de testes.


Link para o WebCode


DCoder



Dcoder é uma ferramenta extremamente eficiente quando se quer programar utilizando um celular já que ele cria um localhost no seu aparelho onde poderá rodar diversas aplicações, não se restringindo as três anteriormente citadas no WebCode
Sua vantagem é que ele é completamente gratuito e você ainda pode criar uma conta no sistema e salvar seus codigos porém para utilizar o Dcoder você precisará estar conectado a internet o tempo todo.


Link para o Dcoder
